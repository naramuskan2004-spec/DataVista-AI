DataVistaAI
An end-to-end data analytics application that transforms raw CSV datasets into actionable insights through interactive visualizations, statistical summaries, correlation analysis, and predictive modeling.

DataVista AI – Smart Data Analysis Dashboard
An AI-powered interactive dashboard built with Streamlit for automated data analysis and visualization of CSV datasets.

Features
Automated EDA and data cleaning
Dynamic visualizations using Plotly
ML-based predictions with Scikit-learn
Query-based assistant for automated insights
Tech Stack
Python | Pandas | Streamlit | Plotly | Scikit-learn

## Run locally

```bash
pip install -r requirements.txt
streamlit run DataVistaAI.py


Dataset
Uses the King County House Sales dataset (available on Kaggle). kc_house_data.csv can use any other csv dataset too
